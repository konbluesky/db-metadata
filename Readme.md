        - 树型组件(依赖元对象)

    - 表单
        - 单选组件(元子段)
        - 多选组件(元子段)
        - 输入组件(元子段)
        - 业务查询组件
        - 开关组件(元子段)
        - 日期组件(元子段)
        - 范围组件(元子段)
