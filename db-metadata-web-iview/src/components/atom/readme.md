# Atom Component
最小业务组件, 原子组件, 或称为带有业务属性的技术组件. 它是不可再分割的. 
- 非Z开头的为基于element-ui封装的
- Z开头为自行封装的

## TextBox - 文本框
文本输入框

## PassBox - 密码框
密码输入框

## TextAreaBox - 文本域
即textarea

## BoolBox - 布尔框
用于输入“是/否”bool值的原子控件

## NumBox - 数字输入框
输入数字

## RadioBox - 单选框

## CheckBox - 复选框

## DropDownBox - 下拉框

## TimeBox - 时间输入框

## DateBox - 日期输入框

## DateTimeBox - 日期时间框

## JsonBox - JSON编辑框

## ImgBox - 图片框

## FileBox - 文件输入框

## MultiImgBox - 多图框
